Class DNav

Responsible for maintaining navigational information, as well as providing navigational capabilities.

Info:

  • Author: Craig Knoblauch

Methods

DNav:new (x, y, z, facing)
Creates and returns a DNav object. Please note, the variables listed here are the only variables assosciated with this class.

Parameters:

  • x The turtle's starting x position
  • y The turtle's starting y position
  • z The turtle's starting z position
  • facing The turtle's starting cardinal direction

Returns:

    A new DNav object

DNav:updateFacing (self, turn_type)
Updates facing attribute in accordance with the type of turn specified. This function must be called BEFORE a turn is executed.

Parameters:

  • self The calling DNav object. Does NOT need to be provided if the ':' operator is used to call the function.
  • turn_type A string. Possibilites are 'right', 'left', and 'around'.

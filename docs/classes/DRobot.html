Methods

DRobot:new (x, y, z, facing)
Creates and returns a DRobot object. The turtle's initial x, y, z position and cardinal facing direction must be known on startup. This information will be passed to DNav to maintain navigational information

Parameters:

  • x The turtle's starting x position
  • y The turtle's starting y position
  • z The turtle's starting z position
  • facing The turtle's starting cardinal direction

Returns:

    A new DRobot object
